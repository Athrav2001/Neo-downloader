# Neo Downloader

A powerful multi-platform download manager built with Kotlin and Jetpack Compose.

## Features

- **Multi-thread Downloads** - Fast downloads with parallel connections
- **HLS Streaming** - Download m3u8 streaming videos
- **Resume Support** - Pause and resume downloads anytime
- **Queue Management** - Organize downloads with queues and schedules
- **Speed Control** - Set download speed limits
- **Proxy Support** - Configure proxy settings
- **Browser Integration** - Capture downloads from in-app browser
- **Dark/Light Themes** - Beautiful neon-themed UI

## Platforms

- Android (Primary)
- Desktop (Windows, Linux, Mac) - Coming soon

## Tech Stack

- Kotlin
- Kotlin Multiplatform
- Jetpack Compose
- Gradle
